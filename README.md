# Registration Form Project

This project is a simple PHP user registration system built using a simplified MVC pattern. Users can register with a username and password, and then view their profile information.

## Features
- User registration form with validation
- Password stored as plain text (for demonstration purposes only)
- User profile page displaying username and password
- Modular MVC-style structure:
  - `controllers/` — Handles form logic
  - `models/` — Handles database interactions
  - `views/` — Contains HTML templates
  - `views/partials/` — Header and footer templates
- PDO used for secure database interactions

## AI Tools
This project was developed with the assistance of ChatGPT to improve coding structure, implement functions, and connect the database successfully.
